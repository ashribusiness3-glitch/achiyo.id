# Achiyo - Handmade Bags & Accessories Website

## 📱 About
Achiyo is a modern, aesthetic website showcasing premium handmade macramé bags and accessories. The website features an interactive product catalog with detailed product information and direct purchasing via Instagram.

## 🎨 Features

### ✨ Key Features:
- **Modern & Aesthetic UI** - Clean, minimalist design with earth tones and elegant typography
- **Responsive Design** - Fully responsive website that works on desktop, tablet, and mobile
- **Interactive Product Modal** - Click on any product to view detailed information
- **Multiple Product Series**:
  - Aurora Series (Midnight Blue)
  - Serenity Series (Crimson, Dusty Pink)
  - Botanica Series (Forest Green)
  - Harmoni Series (Natural)
  - Bohemian Series (Dusty Green)
- **Direct Instagram Integration** - "Buy Now" button redirects to Instagram store
- **Testimonials Section** - Customer reviews and feedback
- **About & Contact Sections** - Brand information and social media links
- **Smooth Animations** - Fade-in effects and hover animations
- **User-Friendly Navigation** - Smooth scrolling navigation menu

## 🚀 How to Use

### 1. **View the Website**
   - Open `index.html` in your web browser
   - Alternatively, use a local server (VSCode Live Server extension recommended)

### 2. **Browse Products**
   - Scroll through the "Our Collection" section
   - Products are displayed in a responsive grid layout
   - Hover over products to see animations

### 3. **View Product Details**
   - Click on any product card or "View Details" button
   - A modal window will open showing:
     - Complete product name and series
     - Detailed description
     - Full specifications
     - Price information
   - Close the modal by clicking the X button or clicking outside

### 4. **Purchase**
   - Click "Beli Sekarang" (Buy Now) button in the product modal
   - This will redirect to Achiyo's Instagram store in a new tab
   - Instagram URL: https://www.instagram.com/achiyo.idn

### 5. **Navigation**
   - Use the navigation menu to jump to different sections:
     - Home (Hero section)
     - Products
     - About
     - Testimonial
     - Contact

## 📝 Note for Developers

### To Customize:
1. **Add New Products**: Edit `productData` object in `js/script.js`
2. **Change Colors**: Update CSS variables in `:root` selector in `css/styles.css`
3. **Modify Text**: Update content in `index.html`
4. **Add Images**: Replace gradient backgrounds with actual image URLs

### To Deploy:
1. Upload all files to a web hosting service
2. Ensure folder structure is maintained (css/, js/ folders)
3. No backend server required - static website only
